Step:
- Download dependencies with command: npm i
- Copy the dotenvtemplate.txt file, place it in genshin_import_api folder and rename to .env
- Open XAMPP Control Panel
- Start Apache and MySQL
- Open terminal
- cd to genshin-import-backend/genshin_import_api
- run command: npm run dev

Extra:
- You can copy dummy.txt SQL data as the data dummy to the database in localhost.
